    Box button

    Box button은 박스 형태로 된 가장 일반적인 버튼입니다.
    화면의 컨텐츠 영역에 배치하여 화면 내 주요한 액션을 실행하는 버튼으로 사용합니다.

    Anatomy

    1. Button container버튼의 형태를 나타냅니다. 버튼의 배경컬러를 통해 속성을 구분할 수 있습니다.

    2. Button label area버튼의 내용을 나타내며, 중앙정렬을 기본으로 합니다.

    3. Icon area (optional)버튼의 내용을 대표하는 아이콘을 좌측에 함께 사용할 수 있습니다. 텍스트와 동일한 색상을 사용합니다.
    Line type에 아이콘을 조합할 경우 J black 컬러의 아이콘 위주로 사용하며, J orange 컬러는 지정된 아이콘만 조합할 수 있습니다.

    Iconography guide 보기

    Spec

    Box Button은 다음과 같이 Large, Medium, Small 사이즈를 제공합니다.

    LargeMediumSmall

    일반적으로 Box Button의 패딩은 16px 입니다.

    예외적으로 가로값은 변경 가능하나, 각 사이즈별 세로값은 변경할 수 없습니다.

    Type

    Box button의 형태와 용도에 따라 type이 구별됩니다.

    Fill (Primary)

    J blackJ orange

    가장 강한 강도로 강조하는 버튼으로, 유저의 다음 행동이 명확할 때 사용합니다.

    Box Button의 기본 형은 J black, J orange 컬러로 배경이 채워진 형태입니다.

    Fill (Secondary)

    successcancelGray

    긍정과 부정의 액션을 수행하는 버튼에는 각각 보조색상 success / cancel를 사용합니다.

    보조 버튼(Secondary Button)에서 Gray 색상 버튼은 Primary Buttons의 보조로 사용합니다.

    Line (Secondary)

    J blackJ orange

    상대적으로 중요도가 낮거나, 한 페이지에 2개 이상의 Secondary Button이 필요할 때 사용합니다.

    States

    Default, Hover, Active, Disabled 4가지의 상태를 갖고 화면의 상태와 유기적으로 작용합니다.

    Fill

    J blackJ orangesuccesscancelGray
    Default Hover Active Disabled

    Line

    J blackJ orange
    Default Hover Active Disabled

    Basic

    Button UI 기본 예시입니다. Box button은 사용 목적에 따라 Primary, Secondary, Success, Cancle 버튼으로 구분하여 사용합니다.

    HTML 
    <button type="button" class="btn medium primaryA">Primary</button>
<button type="button" class="btn medium primaryB">Primary</button>
<button type="button" class="btn medium success">Success</button>
<button type="button" class="btn medium cancel">Cancel</button>
<button type="button" class="btn medium secondary">Secondary</button>
<button type="button" class="btn medium outline-primaryA">Secondary</button>
<button type="button" class="btn medium outline-primaryB">Secondary</button>

    Size

    버튼의 크기를 .large, .mediem, .small을 적용해 변경할 수 있습니다.

    HTML 
    <button type="button" class="btn large primaryA">Large button</button>
<button type="button" class="btn medium primaryA">Medium button</button>
<button type="button" class="btn small primaryA">Small button</button>
<button type="button" class="btn large outline-primaryA">Large button</button>
<button type="button" class="btn medium outline-primaryA">Medium button</button>
<button type="button" class="btn small outline-primaryA">Small button</button>

    Fill button

    버튼의 백그라운드를 .primaryA, .primaryB, .success, .cancel, .gray를 적용해 변경할 수 있습니다.

    HTML 
    <button type="button" class="btn medium primaryA">Primary</button>
<button type="button" class="btn medium primaryB">Primary</button>
<button type="button" class="btn medium success">Success</button>
<button type="button" class="btn medium cancel">Cancel</button>
<button type="button" class="btn medium secondary">Secondary</button>

    Line button

    Line button은 Primary Color 두 가지만 지원합니다.
    기본 변경 클래스를 .outline-*으로 변경해서 버튼에 있는 배경색을 지우고 아웃라인을 추가할 수 있습니다.

    HTML 
    <button type="button" class="btn medium outline-primaryA">Secondary</button>
<button type="button" class="btn medium outline-primaryB">Secondary</button>

    Disabled status

    버튼을 비활성화시키기 위해 버튼 요소에 disabled 속성을 추가합니다.
    비활성화된 버튼은 pointer-events: none을 가집니다.

    HTML 
    <button type="button" class="btn medium primaryA"  disabled>Primary</button>
<button type="button" class="btn medium outline-primaryB"  disabled>Secondary</button>

    Block button

    Bolck button은 Large사이즈만 사용하며, 가로값은 상황에 따라 변경할 수 있습니다.

    HTML 
    <button type="button" class="btn large primaryA block">Primary</button>

    Combination

    버튼에 .ic, .ic-*를 추가하면 오른쪽에 아이콘이 있는 버튼을 사용하실 수 있습니다.

    HTML 
    <button type="button" class="btn medium primaryA ic ic-search">Button</button>
<button type="button" class="btn medium primaryB ic ic-search">Button</button>
<button type="button" class="btn medium success ic ic-search">Button</button>
<button type="button" class="btn medium cancel ic ic-search">Button</button>
<button type="button" class="btn medium secondary ic ic-search">Button</button>
<button type="button" class="btn medium outline-primaryA ic ic-search">Button</button>
<button type="button" class="btn medium outline-primaryB ic ic-search">Button</button>
