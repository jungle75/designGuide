1. Table header표시될 데이터의 제목을 나타냅니다.

2. List목록에는 제목에 해당되는 데이터가 표시됩니다.

Spec

Table은 기본적으로 다음과 같은 간격 및 행간 값을 가지며, 화면 상태에 따라 유동적으로 변경될 수 있습니다.

Type

테이블은 2가지 타입으로 구성되며, 데이터의 내용에 따라 형태가 다르게 보여집니다.

일반적으로 데이터는 1줄로 노출하며, 1줄을 넘어갈 경우 말줄임 표시를 적용합니다. (단, 2줄이상 노출이 필요할 경우 Long text 용법을 활용합니다.)
