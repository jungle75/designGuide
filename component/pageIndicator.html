Page indicator

숫자 혹은 표식(점)을 통해 전체 정보의 양과 현재 사용자가 속한 지점을 안내합니다.

Anatomy

3 6

1. Activated dot사용자가 현재 보고 있는 페이지 입니다.

2. Inactive dot전체 페이지 수를 나타내며, 비활성 도트로 표시됩니다.

3. Background area숫자의 가독성을 높이기 위해 배경과의 구분이 필요한 경우 사용합니다.

4. Current page사용자가 현재 보고 있는 페이지 입니다.

5. Total page전체 페이지 숫자입니다.

Spec

Dot Indicator

Dot은 최대 5개까지 사용가능하며, Center 정렬로 사용합니다.

Number Indicator

3 6

Background area 는 #000000 50%를 고정으로 사용합니다.

Usage

인디케이터는 콘텐츠의 구성 내용에 따라 Dot Indicator, Number Indicator로 분류됩니다.

Dot Indicator

bg : Dark
bg : Light

Dot Indicator의 경우 ui가 배치되는 배경 컬러에 따라 다르게 사용합니다.

Dot Indicator는 최대 5개 까지 사용 가능하고, 5개를 초과할 시 넘버인디케이터를 사용합니다.

Number Indicator

3 6

Number Indicator는 콘텐츠의 개수가 명확하게 표시되어야 하는 경우 사용합니다.

이미지 내부 우측 하단에 겹쳐서 고정으로 사용합니다.
