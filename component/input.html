Input

Input은 문자, 숫자, 기호등을 입력하고 편집할 수 있는 입력 컴포넌트입니다.

Anatomy

1. Label (Optional)사용자가 입력해야하는 내용이 무엇인지 표시됩니다.

2. Text area사용자가 입력한 내용이 표시됩니다.

Spec

Input은 다음과 같이 Default, Large 사이즈를 제공합니다.

Default Large

Input는 기본적으로 Default 사이즈를 사용하며, 로그인 및 주의를 요하는 상황에서 단독으로 사용할 경우 Large 사이즈를 사용합니다.

Input의 패딩은 Default, Large 사이즈 모두 16px 입니다.

예외적으로 가로 값은 변경 가능하나, 각 사이즈별 세로 값은 변경할 수 없습니다.

States

Enabled, Hover, Active, Error, Disabled 5가지의 상태를 갖고 화면의 상태와 유기적으로 작용합니다.

Enabled Hover Active Error Disabled

Usage

Label 유무에 따라 텍스 입력필드가 다르게 보여집니다.

Placeholder usage

Label을 사용하지 않고 플레이스홀더로만 안내할 경우,
입력해야 하는 내용은 입력한 내용을 지워야만 볼 수 있습니다.
입력 필드가 많은 경우, Label을 적용하여 더 명확하게 세부 내용을
전달해 주세요.

Label usage

Label은 입력필드의 좌측에 배치할 수 있으며
입력해야하는 내용의 제목과 같은 의미를 갖습니다.
Label 사용 시 Placeholder 와 내용이 겹치지 않도록 주의가 필요합니다.

Text area

0/100

필요에 따라 Text area의 높이값을 고정하거나, 콘텐츠 높이에 자동으로 맞춰 조정할 수 있습니다.

Basic

Input UI 기본 예시입니다.

HTML 
<div class="inputBox"><input type="text" placeholder="Placeholder"></div>

Label position

<Label>은 입력 그룹 밖에 작성해야 합니다.

HTML 
<label for="a">Label</label><div class="inputBox"><input type="text" id="a" placeholder="Placeholder"></div>

Size

Input에는 Default(34px), Large(40px) 의 두 가지 크기가 있습니다.

HTML 
<div class="inputBox"><input type="text" placeholder="Placeholder"></div>

  <div class="inputBox large"><input type="text" placeholder="Placeholder"></div>

Error / Disabled status

status을 사용하여 입력에 상태(Error, Disabled)을 추가합니다.

HTML 
<div class="inputBox invalid"><input type="text" value="Text Inputting Error"></div>

  <div class="inputBox"><input type="text" placeholder="-" disabled></div>

With icon

Input UI에 아이콘을 추가로 배치할 수 있습니다.

HTML 
<div class="inputBox">
    <input type="text" value="2024.07.01"><i class="date"></i>
  </div>

  <div class="inputBox large">
    <input type="text" value="2024.07.01"><i class="date"></i>
  </div>

With button

HTML 
<div class="formCol">
    <div class="inputBox"><input type="text" placeholder="Placeholder"></div>
    <button type="button" class="btn small primaryA ic ic-search">Button</button>
  </div>

  <div class="formCol">
    <div class="inputBox large"><input type="text" placeholder="Placeholder"></div>
    <button type="button" class="btn medium primaryA ic ic-search">Button</button>
  </div>

      <select>

      </select>

    <div class="inputBox large"><input type="text" placeholder="Placeholder"></div>
