    Text button

    주로 메인 액션에 대한 부가적인 행동이나,
    한 페이지 내에 다양한 기능을 담아야 할 경우 Text button을 사용합니다.

    Anatomy

    1. Icon area (optional)버튼의 내용을 대표하는 아이콘을 좌측에 함께 사용할 수 있습니다.
    기본적으로 텍스트와 동일한 색상을 사용하며 Blue, Red 컬러는 지정된 아이콘만 조합할 수 있습니다. Iconography guide 보기

    2. Text area버튼의 내용을 나타내며, 중앙정렬을 기본으로 합니다.

    3. link (optional)일반 텍스트와의 구분을 위해 사용합니다. 텍스트와 동일한 색상을 사용합니다.

    States

    Default, Hover, Active, Disabled 4가지의 상태를 갖고 화면의 상태와 유기적으로 작용합니다.

    J blackBlueRedGray
    Default Hover Active Disabled

    [Normal - Black] : Text Button 중 일반적인 액션 버튼으로 사용합니다.

    [Positive - Blue / Negative - Red] : 긍정과 부정 액션에 대해 사용합니다.

    [Retreative - Gray] : Text Button 중 'Cancel/Close' 등 현재 화면에서 후퇴하는 액션에 대해 사용합니다.

    Basic

    Text button UI 기본 예시입니다.

    HTML 
    <button type="button" class="textBtn">Text Button</button>
  <button type="button" class="textBtn blue">Text Button</button>
  <button type="button" class="textBtn red">Text Button</button>
  <button type="button" class="textBtn gray">Text Button</button>

    Disabled status

    status을 사용하여 입력에 상태(Disabled)를 추가합니다.

    HTML 
    <button type="button" class="textBtn"  disabled>Text Button</button>

    Combination

    텍스트 버튼에 .arrow를 추가하면 오른쪽에 링크아이콘을 사용할 수 있고, .ic, .ic-*를 추가하면 왼쪽에 아이콘을 사용하실 수 있습니다.

    HTML 
    <button type="button" class="textBtn ic ic-search">Text Button</button>
  <button type="button" class="textBtn blue ic ic-search">Text Button</button>
  <button type="button" class="textBtn red ic ic-search">Text Button</button>
  <button type="button" class="textBtn gray ic ic-search">Text Button</button>

    CSS

    해당 컴포넌트만 적용 가능한 CSS 파일입니다.

    resest.css 가 없을 경우 깨질 수 있습니다. 컴포넌트 메인페이지에서 reset.css 파일을 다운로드 받아서 사용하세요.
