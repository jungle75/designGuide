Tabs

탭(Tabs)은 페이지 내 유사한 정보를 그룹핑하여 콘텐츠를 정리할 때 사용합니다.

1. Text area

탭의 내용을 표시하는 영역입니다. 색상 등을 사용하여 선택한 탭을 강조 표시할 수 있습니다.

2. Selected border

활성 탭과 비활성 탭을 구분하기 위해 다른 색상을 적용할 수 있습니다.

3. Box tabs

HTML JavaScript 
<div class="tabs textType" id="tabs">
  <a href="#" class="act">tab 1</a>
  <a href="#">tab 2</a>
  <a href="#">tab 3</a>
  <a href="#">tab 4</a>
  <a href="#">tab 5</a>
</div>
const tabs = document.querySelectorAll('.tabs a');
tabs.forEach(tab => {
  tab.addEventListener('click', () => {
    // 모든 탭에서 'act' 클래스 제거
    tabs.forEach(t => t.classList.remove('act'));
    // 클릭된 탭에 'act' 클래스 추가
    tab.classList.add('act');
  });
});
