States

Icon과 Text 조합 타입으로 dafault/active 상태를 보여줍니 다.

Check box + Text type

선택 영역에서 하나 또는 여러가지를 동시에 선택할 때 사용합니다.

Radio button + Text type

선택 영역에서 단일 상태를 선택하는 데 사용합니다. 선택 항목이 너무 많지 않아야 합니다.

Toggle button + Text type

텍스트 조합 타입 off
텍스트 조합 타입 on

On/Off를 설정할 때 사용합니다.

Check box

HTML 
<label class="checkbox" for="check1">
  <input type="checkbox" name="check1" id="check1" hidden>
  <i></i>
</label>

<label class="checkbox" for="check2">
  <input type="checkbox" name="check2" id="check2" hidden checked>
  <i></i>
  <span>텍스트 조합</span>
</label>
