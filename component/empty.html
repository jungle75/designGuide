Empty

제공된 데이터가 없을 때 표시됩니다.

Anatomy

Data Not Found

No Data 텍스트는 데이터가 존재하지 않을 때 "No data to display" 같은 텍스트를 보여주는 옵션입니다.

1. Image area(Optional)사용자의 상황을 이해하는 데 도움이 되는 용도로만 사용합니다.

2. Contents area현재 상황에 따른 사용자의 예상 동작을 간략히 설명합니다.

3. Button area(Optional)사용자의 예상 동작을 수행할 수 있도록 작업을 프롬프트하는 버튼을 제공합니다.

</div>
