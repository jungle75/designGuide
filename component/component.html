Component

컴포넌트(Component)는 사용자 인터페이스를 구성하는 요소입니다.
목적에 따라 Action, Input, Containmnet, Status, Selection, Navigation등의 범주로 구성하며 조합하여 사용할 수 있습니다.

다운로드

컴포넌트를 적용하기 위해서는 개발 환경에 CSS 파일 업데이트가 필요합니다.

reset.css를 포함한 전체 컴포넌트를 다운 받을 경우 style.min.css (경량화 버전)

필요한 컴포넌트만 다운 받을 경우 추가 적용이 필요한 reset.css

Action

Box button

Box button은 가장 일반적인 버튼입니다.
화면의 컨텐츠 영역에 배치하여 화면 내 주요한 액션을
실행하는 버튼으로 사용합니다.

Text button

주로 메인 액션에 대한 부가적인 행동이나, 한 페이지
내에 다양한 기능을 담아야 할 경우 Text Button을
사용합니다.

Input

Input

Input은 문자, 숫자, 기호등을 입력하고 편집할 수 있는
입력 컴포넌트입니다.

Empty

제공된 데이터가 없을 때 표시됩니다.

Containment

Table

Table에는 데이터가 표시되며 데이터를 조회 / 입력
할 수 있습니다.

Popup

Popup은 사용자에게 중요한 정보를 전달하거나
사용자의 선택을 요청하기 위해 나타나는 모달 형태의
컴포넌트입니다.

Status

Pagination

Pagination을 사용하여 긴 목록을 여러 페이지로
나눌 수 있으며, 한 번에 한 페이지만 로드됩니다.

Page indicator

숫자 혹은 표식(점)을 통해 전체 정보의 양과
현재 사용자가 속한 지점을 안내합니다.

Selection

Dropdown

선택 가능한 옵션 개수가 다수일 경우, 드롭다운 메뉴를
사용해 값을 노출합니다.

Selection controls

Checkbox, Radio, Toggle 등 옵션을 선택하는
기능의 UI를 정의합니다.

Navigation

GNB (Global Navigation Bar)

GNB는 수평으로 화면 상단에 위치한 탐색메뉴입니다.

SNB (Side Navigation Bar)

SNB는 수직으로 화면 좌측에 위치하며, 사용자가
세부항목에 빠르게 접근할 수 있도록 도와줍니다.

FNB (Foot Navigation Bar)

웹사이트 내 모든 페이지 하단에 위치하는
컴포넌트입니다.

Tabs

탭(Tabs)은 페이지 내 유사한 정보를 그룹핑하여
콘텐츠를 정리할 때 사용합니다.
