Iconography

아이콘은 사용성 기준으로 제작되었습니다.
일관성을 유지하기 위해서는 가이드라인 규칙을 준수해서 사용해야 합니다.

Anatomy

1. Size UI 환경과 상황에 따라 16px, 24px, 48px 사이즈의 아이콘을 사용합니다.

2. Guide 24 사이즈 아이콘 기준, 상하좌우 2px 사이즈의 여백을 제외한 가이드라인에 벗어나지 않도록 비율에 맞춰 제작합니다.

Icon

24px 기준 아이콘의 경우 라인의 두께는 2~2.5로 제작하고 Round 값은 1~2로 지정합니다.

기본적인 아이콘은 J Black으로 제공되며 J orange, Blue, Red 컬러의 아이콘은 지정된 특정 아이콘만 제공됩니다.

SVG Download 시 J Black, J orange, Blue, Red 총 4가지 컬러의 아이콘 Set을 다운받을 수 있습니다.

Design guide

아이콘을 실제 사용할 경우 컬러와 간격의 가이드를 준수합니다.

Colors

  • J Black (#111111)
  • Gray-800 (#555555)
  • Gray-500 (#9E9E9E)

기본적인 아이콘은 J Black으로 제공되며, 환경에 따라 Color guide에 맞춰 커스텀 될 수 있습니다.

Space

아이콘 사용 시 간격은 동일해야하며 24px 버튼 기준 아이콘 사이 간격은 최소 16px입니다.

Combination usage

아이콘은 버튼, 인풋 UI와 조합하여 사용할 수 있습니다.
